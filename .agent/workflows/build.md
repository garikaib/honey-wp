# Build Theme Assets

## Development Mode (with hot reload)
// turbo
```bash
cd /home/garikaib/Documents/sites/honey-wp/wp-content/themes/honeyscroop-theme
npm run dev
```

## Production Build
// turbo
```bash
cd /home/garikaib/Documents/sites/honey-wp/wp-content/themes/honeyscroop-theme
npm run build
```

## Deployment
After building, use the [Sync Workflow](file:///.agent/workflows/sync.md) to push changes to production.
